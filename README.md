# moniruzzaman Portfolio

![Portfolio Screenshot](https://dev-moniruzzaman.vercel.app/moniruzzaman.png)

## Overview

This repository contains the **personal portfolio of Yousuf**, a **NYC-based web developer** specializing in building **SEO-friendly, responsive websites for small businesses**. The portfolio showcases projects, skills, and services, highlighting expertise in **React, Next.js, Tailwind CSS**, and modern web development best practices.

The project is built with **Next.js 13** using the App Router for server-side rendering and static site generation, ensuring **fast performance, SEO optimization, and accessibility**. It includes structured data, dark/light theme support, and a clean, professional design.

---

## Features

- **SEO-Friendly**: Meta tags, Open Graph, Twitter cards, keywords, and structured JSON-LD for better search visibility.  
- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices using Tailwind CSS.  
- **Dark/Light Mode Toggle**: Users can switch between themes; preference is saved for future visits.  
- **Project Showcase**: Highlighted projects with images, descriptions, and technologies used.  
- **Contact & Social Links**: Easy ways to reach out via email or LinkedIn.  
- **Modern Tech Stack**: Next.js, React, Tailwind CSS, optimized images, and accessibility best practices.

---

## Tech Stack

- **Frontend**: React.js, Next.js 13  
- **Styling**: Tailwind CSS  
- **Fonts**: Geist & Geist Mono (Google Fonts)  
- **SEO Optimization**: Meta tags, OG/Twitter cards, structured data (JSON-LD)  
- **Images**: Next.js `Image` component for optimized loading
